## 🎯 **What is this?**
**Advanced Streamlit ML app combining:**
- **Multi-Addiction Risk ML** (Smoking, Alcohol, Social Media, Junk Food prediction)
- **Ayurveda Triguna ML** (Sattva/Rajas/Tamas prediction + classification)
- **Bhagavad Gita IKS Interventions** (personalized verses + Yoga/Pranayama)
- **Reinforcement Learning RL** (Q-Learning optimizes interventions)
- **Guided Meditation Tab** (live second-wise timer + brain dump)
- **IKS Chatbot** (Gita-based motivation)
- **Real-time Risk Dashboards** (RF + SHAP-ready, Plotly)

**Key:** Production-ready. **NEW:** Trains ML model on your personal usage data!

## 🚀 **1-Click Deploy**
```bash
git clone <repo>
cd iks-ds
python -m venv venv  # Recommended
# Windows: venv\Scripts\activate
# Linux/Mac: source venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## 🏗️ **Tech Stack**
```
Frontend: Streamlit 1.38 + Plotly 5.22 + Glassmorphism CSS
ML: scikit-learn 1.5 (RF, pipelines), joblib (user-trained models)
RL: Custom Q-Learning (config.py: lr=0.1, γ=0.95)
Data: Pandas, iks_interventions.json (Gita by addiction/guna)
Config: config.py (API stubs, RL params)
Deployment: Production-ready (pinned deps, caching)
```
